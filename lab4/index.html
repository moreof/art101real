Lab 4 - Pseudocoding and Problem-solving

Challenge

The challenge for this lab was to write pseudocode for an everyday task and for a simple computer game. The pseudocodes for each then must be incorporated into the Lab 4 javascript file as comments, and to then add that js code to the index html file and design a page to host this lab and our pseudocode, incorporating any images necessary.

Problems

I initially had some issues with the actual practice of pseudocoding, and specifically in figuring out how detailed my instructions should get, or how to reflect two actions that are meant to occur at the same time. My main challenge, as is probably still visible on this site, is the alignment of my pseudocode on the webpage. I spent a couple of hours trying different codes I could find online and trying to incorporate feedback from an AI, but I can't figure out how to specifically target the pseudocode and align it within normal page boundaries.

Reflection

Overall, I think this assignment went well. I wish I could've figured out that formatting, but it is what it is. I do think the process of pseudocoding is helpful in that it helps to think of programming/coding as a language, and forces us to start incorporating its "grammar". I think some of my pseudocode doesn't perfectly correspond to how actual code would work, particularly in the randomized selection.
